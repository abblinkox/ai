Mappen innehåller ett dataset som innehåller bilder på 2 olika bilar, en Ferrari och en Lamborghini
I mappen:
annotations ligger det labels som txt-filer, detta för att man ska kunna träna yolo med darknet om man vill.
annotations_xml ligger det labels som xml filer, detta för att man ska kunna träna med Keras.
images, ligger bilderna som ska användas för träning
testing_annotations_xml, ligger annotations som xml för validering/testing
testing_images, ligger bilderna som ska användas för validering/testing
